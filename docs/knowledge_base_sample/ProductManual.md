# Product Manual

## 1. 产品概述

SupportHub 是一套企业服务支持平台，提供账号管理、工单提交、知识库查询、审批流转和通知提醒等能力。员工可以通过该平台快速发起问题咨询，客服或 IT 支持人员可以通过工单系统跟进处理。

## 2. 用户角色

### 2.1 普通员工
- 查询常见问题；
- 提交工单；
- 查看工单状态；
- 接收通知和处理结果。

### 2.2 客服/支持人员
- 查看待处理工单；
- 回复用户；
- 更新工单状态；
- 添加处理备注；
- 升级高风险问题。

### 2.3 管理员
- 维护知识库；
- 配置工单分派规则；
- 处理权限申请；
- 查看系统统计报表；
- 管理公告和通知模板。

## 3. 核心模块

### 3.1 登录与账号中心
- 支持账号密码登录；
- 支持忘记密码；
- 支持账号锁定与解锁；
- 支持基础个人信息维护。

### 3.2 知识库中心
- 支持 FAQ 搜索；
- 支持 SOP 查询；
- 支持产品手册浏览；
- 支持历史工单检索。

### 3.3 工单中心
- 支持用户创建工单；
- 支持按分类自动分派；
- 支持状态流转；
- 支持上传截图和附件；
- 支持备注和处理记录。

### 3.4 通知中心
- 支持邮件通知；
- 支持审批提醒；
- 支持工单状态变更提醒；
- 支持系统公告。

## 4. 常见操作

### 4.1 提交工单
1. 登录平台；
2. 点击“新建工单”；
3. 填写标题、描述、分类和附件；
4. 提交后等待系统分派。

### 4.2 查询工单
1. 打开“我的工单”；
2. 输入工单编号；
3. 查看状态、处理人和备注；
4. 需要时继续补充信息。

### 4.3 搜索知识
1. 在搜索框输入问题；
2. 系统返回 FAQ、SOP 或历史案例；
3. 查看引用来源；
4. 根据建议执行操作。

## 5. 系统规则

- 敏感操作必须经过人工确认；
- 退款、权限和账号相关操作应保留审计记录；
- 历史工单和知识库文档按版本管理；
- 高频问题优先沉淀为 FAQ。

## 6. 常见问题

### 6.1 找不到工单怎么办？
请确认工单编号是否正确，并检查当前账号是否有查看权限。

### 6.2 为什么无法提交附件？
请检查附件大小是否超限，文件格式是否受限。

### 6.3 为什么知识库没有答案？
可能是检索关键词不准确，也可能是知识库尚未收录该问题。建议补充更多描述或提交工单。
