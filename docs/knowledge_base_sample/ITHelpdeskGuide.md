# IT Helpdesk Guide

## 1. 邮箱无法登录

### 可能原因
- 密码错误；
- 账号过期；
- MFA 验证失败；
- 邮箱服务临时异常。

### 处理步骤
1. 确认账号是否有效；
2. 检查密码是否正确；
3. 尝试重新登录并刷新验证码；
4. 检查是否开启二次验证；
5. 若仍失败，升级给 IT 支持组。

## 2. VPN 无法连接

### 可能原因
- 客户端版本过旧；
- 账号权限不足；
- 网络环境异常；
- 证书过期。

### 处理步骤
1. 确认外网是否正常；
2. 检查 VPN 客户端版本；
3. 检查账号是否开通 VPN 权限；
4. 重新导入证书；
5. 若仍失败，提交网络工单。

## 3. 打印机无法打印

### 可能原因
- 打印机离线；
- 驱动损坏；
- 队列卡住；
- 网络打印机地址错误。

### 处理步骤
1. 检查设备是否在线；
2. 重启打印服务；
3. 清理打印队列；
4. 重新安装驱动；
5. 测试打印一页。

## 4. 公司电脑启动缓慢

### 可能原因
- 启动项过多；
- 磁盘空间不足；
- 杀毒扫描占用；
- 系统更新未完成。

### 处理步骤
1. 检查磁盘剩余空间；
2. 禁用不必要启动项；
3. 清理临时文件；
4. 检查是否存在更新卡顿；
5. 若持续异常，升级给终端支持。

## 5. 软件无法安装

### 可能原因
- 账号权限不足；
- 安装包损坏；
- 系统版本不兼容；
- 安全策略限制。

### 处理步骤
1. 确认安装包来源；
2. 检查管理员权限；
3. 查看系统兼容性；
4. 尝试以管理员身份安装；
5. 若仍失败，联系 IT 支持。

## 6. 远程桌面无法连接

### 可能原因
- 目标主机离线；
- 防火墙拦截；
- 账号无远程登录权限；
- 端口未开放。

### 处理步骤
1. 确认目标主机在线；
2. 检查网络连通性；
3. 确认账号有远程权限；
4. 检查防火墙与端口配置；
5. 若仍失败，转交网络或系统管理员。

## 7. 浏览器访问内网系统异常

### 可能原因
- 缓存问题；
- 浏览器版本过旧；
- 证书不受信任；
- 网络代理配置错误。

### 处理步骤
1. 清理浏览器缓存；
2. 尝试无痕模式；
3. 检查浏览器版本；
4. 检查系统证书；
5. 若仍失败，记录错误截图并建单。
